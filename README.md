# Restaurant Ordering System – Monorepo

Monorepo ini berisi dua komponen utama:

- **Backend** — FastAPI + SQLAlchemy + MySQL
- **Frontend** — HTML, CSS, dan JavaScript (fetch API)

Sistem ini menyediakan fitur:

- Manajemen Pelanggan
- Manajemen Meja
- Manajemen Menu
- Reservasi Meja
- Pemesanan (Order & Order Item)
- Laporan Harian
